Robber Go

Go Go!

Game 1- First Vision

All Made By Processing

The first project I would like to describe is a game I designed for IAT167. The game is called Robber Go Go Go! This is an avoid obstacles/shooter game. There are many enemies which are cars, as obstacles for the Player. Player will be a robber and the boss enemy will be a policeman. Player needs to avoid those cars to avoid being hurting by cars or being caught by the policeman. For the First Vision, all elemnts are made by Processing Coding and first idea is achieved by Processing, like the SpeedUp Bag and FirstAid Bag.

Game 2- Final Vision

Processing + AI + Sound

There are 3 levels in total for the game and the difficulty will become more difficult as the level is upgraded.This is an individual project and The original intention for this game is designed for those people who like the sensation that stimulates the heartbeat and the sense of speed. The audience will be those people aged between 10 to 30. I learned the Processing in IAT167 so I applied the coding technology that learned in course to the game as well as the design ability becauses I made the role and environment for the game using AI and developed the sound through Audacity.
